1. Сбор персональных данных

Наш сервисный центр собирает персональные данные клиентов для оказания услуг по ремонту и обслуживанию техники.

2. Какие данные собираются

ФИО, контактный телефон, адрес, данные паспорта (для гарантийного обслуживания).

3. Использование данных

Данные используются исключительно для оформления заказов и связи с клиентом.

4. Защита данных

Все данные хранятся в защищенной базе данных и не передаются третьим лицам.
